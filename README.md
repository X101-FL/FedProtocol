## Requirements

- `python >= 3.8`
- `pycrypto >= 2.6.1`: needed for Private Set Intersection (PSI)
- `pytest >= 6.2.4`: needed for union test
- (Optional) `black`: better code formatter [Installation](https://pypi.org/project/black/)
